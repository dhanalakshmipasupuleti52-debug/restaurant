Chilli Restaurant Website

This project is a simple restaurant website
developed using HTML and CSS.

The main goal of this project is to design
a clean and attractive restaurant menu page
that displays food items and prices clearly.

The website contains a restaurant header,
menu section and different food items like
Biryani, Chilli Chicken and Butter Chicken.

Each item includes an image, short description
and price to make the menu easy to read
for users visiting the website.

Through this project, I practiced basic
web development concepts such as layout
design, image placement and styling with CSS.

This project helped me improve my
HTML structure and CSS styling skills
while building a simple real-world website.
